# 🚀 JavaScript Mini Projects – Real Life Features

This repository contains a collection of **small JavaScript projects inspired by real-life UI interactions**.  
Each project focuses on improving **DOM manipulation, event handling, and interactive UI behavior using Vanilla JavaScript**.

These projects helped me practice **core JavaScript concepts by building instead of only watching tutorials.**

---

## 📌 Projects Included

### 1️⃣ Add Friend Feature
A simple **social media style interaction** where clicking the button changes the user status from **"Not Friends" to "Friends"**.

**Concepts Used**
- DOM Manipulation
- Click Events
- Dynamic Text Updates

---

### 2️⃣ Custom Cursor
A **custom cursor animation** that replaces the default mouse pointer and follows the mouse movement.

**Concepts Used**
- `mousemove` event
- CSS positioning
- Dynamic cursor effects

---

### 3️⃣ Instagram Love Feature ❤️
Double-clicking on the image shows a **heart animation**, similar to the like feature used on Instagram.

**Concepts Used**
- `dblclick` event
- CSS animations
- Timed element visibility

---

### 4️⃣ Mousemove Image Interaction
An image dynamically **moves or reacts based on mouse position**, creating an interactive hover effect.

**Concepts Used**
- Mouse coordinates
- Event listeners
- Element positioning

---
### 5️⃣ Instagram Story Feature

A simple Instagram-style story feature where users can click on profile images to view stories. This project simulates the basic behavior of Instagram stories using JavaScript.

**Concepts Used**
- DOM manipulation
- Click events
- Dynamic image display
- Object data
- Array data handling

### 📱 Interactive UI Experiments
Small UI experiments that simulate **real-world website interactions** and improve frontend development skills.
